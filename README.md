# Solana SPL token claimer

## Requirements

- Rust
- Solana
- Anchor
- Yarn

## Overview

Uses a bitmap to mark an indexed claim as used.

Each claim has:

- claim index  
- source token account (on Solana, each balance each SPL token for each owner occupied its own account)  
- amount
